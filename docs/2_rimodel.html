R

First: Save Model (like Stata’s est store)

multi2 <- lmer(stfdem ~ polintr + nwsptot + gndr + (1|cntry), data = ess, REML = FALSE)

Then: Inspect model

tidy(multi2)
glance(multi2)
icc(multi2)
